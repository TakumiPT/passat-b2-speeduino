PRE-FLIGHT CHECKS:
□ Verify CLT reads correctly (compare to known thermometer)
□ Verify TPS reads 0% closed, ~100% WOT
□ Verify MAP reads ~100 kPa at key-on, ~30 kPa at idle
□ Check for fuel leaks at throttle body
□ Check all electrical connections secure

FIRST START PROCEDURE:
1. Turn key ON, wait for fuel pump prime (2-3 seconds)
2. Check for fuel leaks
3. Crank engine, should start within 2-3 seconds
4. Monitor AFR on gauge/laptop
5. Expected idle AFR: 14.5-15.0 (slightly lean OK)
6. Expected warm-up AFR: 13.5-14.5 (rich from WUE)

ROAD TEST PROCEDURE:
1. Let engine fully warm up (90°C+ CLT)
2. Drive gently at first, monitoring AFR
3. Test light acceleration - should be smooth
4. Test moderate acceleration - no hesitation
5. Test WOT briefly - AFR should drop to 12.5-13.0
6. Monitor for any knock/ping sounds

EXPECTED IMPROVEMENTS:
- No more intake backfires during acceleration
- Smoother throttle response
- Better cold start performance
- Proper warm-up enrichment curve
