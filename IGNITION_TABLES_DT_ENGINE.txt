KNOCK DETECTION:
================

What knock sounds like:
- Metallic rattling/pinging sound
- Like marbles in a can
- Occurs under load (acceleration, uphill)
- More noticeable at 2000-3000 RPM under load

If you hear knock:
1. IMMEDIATELY reduce throttle
2. Go back to previous (safer) timing table
3. Or reduce advance by 2-4° in the knock area
4. Test again

Knock can damage your engine! When in doubt, use less advance.


TUNING PROCEDURE:
=================

STEP 1: Start with 91 RON table
- Enter the table in TunerStudio
- Burn to ECU
- Test drive gently

STEP 2: Test for knock
- Drive in various conditions:
  * City (part throttle)
  * Highway (cruise)
  * Uphill (heavy load)
  * Acceleration (WOT)
- Listen carefully for knock

STEP 3: If NO knock detected
- Advance to 95 RON table
- Repeat testing

STEP 4: If knock detected
- Stay with current table OR
- Reduce advance in problem areas by 2°

STEP 5: Fine-tune specific areas
- If knock only at WOT: Reduce 100 kPa row by 2-4°
- If knock at cruise: Reduce 50-66 kPa rows by 2°
- If knock everywhere: Go back to previous table

FUEL OCTANE REFERENCE:
======================

Europe (RON):
- 91 RON = Regular
- 95 RON = Premium (most common)
- 98 RON = Super Premium
- 100+ RON = Race fuel

USA (AKI = (RON+MON)/2):
- 87 AKI ≈ 91 RON
- 91 AKI ≈ 95 RON
- 93 AKI ≈ 98 RON

Your DT engine (9.0:1 compression) is designed for 91 RON minimum.
95 RON is optimal for performance and economy.
98 RON is overkill but safe if available.
