## Electronic Ignition (Speeduino) vs Mechanical Distributor

### ✅ GAINS (What You Get)

| Benefit | Explanation | Real-World Impact |
|---------|-------------|-------------------|
| **Perfect timing everywhere** | Speeduino calculates optimal timing for every RPM + load combination | +5-10% more power, +5-15% better fuel economy |
| **Cold start timing** | Can advance timing when cold for easier starting | No more hard starts in winter |
| **Knock protection** | Can add knock sensor later, auto-retard if knock detected | Engine protection, can run higher compression |
| **No mechanical wear** | No springs, weights, or vacuum diaphragm to wear out | Timing stays perfect forever |
| **Tunable** | Change timing curve in 5 minutes on laptop | Optimize for your fuel, altitude, modifications |
| **Rev limiter** | Cut spark at redline | Engine protection |
| **Launch control** | Retard timing for anti-lag/two-step | If you want to have fun 😄 |
| **Data logging** | See actual timing in logs | Easy troubleshooting |
| **No vacuum leaks** | No vacuum hose to distributor | One less thing to fail |
| **Consistent** | Same timing every time | Smoother idle, better drivability |

---

### ❌ LOSSES (What You Give Up)

| Downside | Explanation | How Bad Is It? |
|----------|-------------|----------------|
| **Complexity** | More wires, more settings | One-time setup, then forget it |
| **Timing light needed** | Must calibrate trigger angle | €30-50 tool, use once |
| **If Speeduino dies = no spark** | Mechanical works without computer | Rare - Speeduino is reliable |
| **Learning curve** | Need to understand timing tables | You already learned fuel tables! |
| **IPO inspector questions** | "What's this?" | Explain it's electronic ignition |

---

### 📊 Direct Comparison: Your Engine

| Condition | Mechanical Distributor | Speeduino Electronic |
|-----------|----------------------|---------------------|
| **Cold idle (-10°C)** | ~18° (fixed by vacuum) | 22-25° (optimized for cold) |
| **Warm idle (80°C)** | ~18° | 18° (same) |
| **Cruise 2500 RPM, light load** | ~28-32° (vacuum + centrifugal) | 36-40° (optimized for economy) |
| **WOT 3000 RPM** | ~28° (centrifugal only) | 26-28° (safer, no knock) |
| **WOT 5000 RPM** | ~32° (max centrifugal) | 32-33° (matched) |
| **High RPM 6000+** | ~32° (stays max) | 30-31° (retards for safety) |
| **Knock detected** | Nothing (keeps advancing) | Auto-retard 5-10° |

---
