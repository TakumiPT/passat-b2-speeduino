## When You Get New DRV8825:

### ⚠️ CRITICAL - Set Current Limit BEFORE Installing!

```
                    ┌─────────────────┐
                    │    DRV8825      │
                    │                 │
                    │   ┌───┐         │
                    │   │POT│ ← Set this FIRST!
                    │   └───┘         │
                    │                 │
                    └─────────────────┘
```

### Current Limit Formula for DRV8825:
```
Vref = Current_Limit × 0.5

For VW IAC (~0.5A):
Vref = 0.5 × 0.5 = 0.25V
```

### How to Set:
1. **DON'T plug into Speeduino yet!**
2. Connect module to 12V and GND only
3. Measure voltage at trimpot (Vref to GND)
4. Adjust to **0.25-0.35V** (for 0.5-0.7A)
5. Then install in Speeduino

## 📊 DRV8825 Output Voltages Explained

### Your readings: 5V to 1V on A1, A2, B1, B2

This means **the driver is working!** ✅

---

### What's happening:

| Output | Function |
|--------|----------|
| A1, A2 | Coil A (one winding) |
| B1, B2 | Coil B (other winding) |

The DRV8825 energizes coils in a **pattern** to make the stepper move:

```
Step 1: A1=HIGH, A2=LOW,  B1=LOW,  B2=LOW
Step 2: A1=LOW,  A2=LOW,  B1=HIGH, B2=LOW
Step 3: A1=LOW,  A2=HIGH, B1=LOW,  B2=LOW
Step 4: A1=LOW,  A2=LOW,  B1=LOW,  B2=HIGH
(repeats)
```

---

## ✅ What 5V to 1V means:

| Reading | Meaning |
|---------|---------|
| **~5V** | Coil is energized (HIGH) |
| **~1V** | Coil is off or PWM chopping |
| **Varies** | Normal! Depends on current step position |
